# Data Indexing and Selection

```python
import pandas as pd
data = pd.Series([0.25, 0.5, 0.75, 1.0],
                 index=['a', 'b', 'c', 'd'])
data
```

```python
data['b']
```

```python
'a' in data
```

```python
data.keys()
```

```python
list(data.items())
```

```python
data['e'] = 1.25
data
```

```python
# slicing by explicit index
data['a':'c']
```

```python
# slicing by implicit integer index
data[0:2]
```

```python
# masking
data[(data > 0.3) & (data < 0.8)]
```

```python
# fancy indexing
data[['a', 'e']]
```
